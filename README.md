# Random Key Generator (C++)

A simple console-based **Random Key Generator** written in C++.
The program generates random keys based on user-defined parameters such as the number of keys, number of words per key, and the number of characters per word.

This project was built as a practice exercise for learning **C++ fundamentals**, including functions, enums, modular design, and random number generation.

## Features

* Generate multiple random keys
* Customizable key structure
* Support for different character types:

  * Capital letters (A–Z)
  * Small letters (a–z)
  * Symbols
  * Alpha-numeric (letters + numbers)
* User-defined:

  * Number of keys
  * Number of words per key
  * Number of characters per word
* Clean modular function-based design
